SMS Webhook Interface

This page is the designated endpoint for our SMS gateway. It will receive incoming text messages, parse the data, and create new leak reports in the system.

Webhook URL: https://api.aquanour.ma/v1/sms/receive

Method: POST

Expected Payload: JSON object containing 'from', 'timestamp', and 'message' fields.
